= Analytics Rails

Adds a simple helper to create the google analytics include tag.

= Install

Put this line in your Gemfile:
  gem 'analytics-rails'
  
Then bundle:
  $ bundle

= Usage
  
In your layout add a line like this in your head:
  = google_analytics_include_tag 'your-id'
